# Roadmap

## Critical deadline

**EU AI Act Article 50 enforcement: 2 August 2026** — approximately 5 months away.

## Phase 1: Specification and core tooling (current)

- [x] APP v1.0 specification draft
- [x] JSON Schema for metadata validation
- [x] Documentation website
- [x] White paper
- [ ] TypeScript SDK (`@ai-provenance-protocol/sdk`)
- [ ] Validator CLI (`npx app-validator validate <file>`)
- [ ] Specification finalisation based on community feedback

## Phase 2: Reference implementation and verification

- [ ] Reference implementation (first conformant platform)
- [ ] Public verification endpoint
- [ ] Python SDK (`ai-provenance-protocol`)
- [ ] Conformance test suite
- [ ] Implementation guide with real-world examples

## Phase 3: Community and ecosystem

- [ ] Integration guides for common platforms (WordPress, Shopify, Next.js)
- [ ] Middleware packages (Express.js, FastAPI, Next.js)
- [ ] 3+ independent implementations
- [ ] Conference presentations and blog posts
- [ ] IETF Internet-Draft or W3C Community Group Note

## Phase 4: Maturity and governance

- [ ] Steering committee formation
- [ ] APP v1.1 based on community feedback
- [ ] C2PA interoperability specification
- [ ] "APP Conformant" certification programme
- [ ] Enterprise adoption toolkit
