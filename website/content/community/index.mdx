# Community

The AI Provenance Protocol is an open standard built in the open. We welcome contributions from developers, organisations, regulators, and anyone interested in AI content transparency.

## Get involved

| Channel | Purpose |
|---------|---------|
| [GitHub Repository](https://github.com/AI-Provenance-Protocol/ai-provenance-protocol) | Source code, specification, and issue tracking |
| [GitHub Discussions](https://github.com/AI-Provenance-Protocol/ai-provenance-protocol/discussions) | Questions, ideas, and community discussion |
| [GitHub Issues](https://github.com/AI-Provenance-Protocol/ai-provenance-protocol/issues) | Bug reports and feature requests |

## How to contribute

- **Specification feedback**: Open an issue or discussion on GitHub
- **Documentation improvements**: Submit a pull request
- **SDK contributions**: Follow the [Contributing Guide](/community/contributing)
- **Implementations**: Build APP support into your platform and share your experience

## Adopters

Are you implementing APP? We'd love to hear about it. Open a GitHub Discussion or issue to be listed as an early adopter.
